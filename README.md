# did-sdk-go

Introduction
------------
Provides general purpose operations for MetaBlox according to W3C spec.


Installation and usage
----------------------

The import path for the package is *gopkg.in/yaml.v3*.

To install it, run:

    go get github.com/metablox/did-sdk-go
